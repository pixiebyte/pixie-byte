# Code of Conduct

## The short version

Be kind, be helpful, be respectful. We're building a community where everyone — regardless of experience level, background, or identity — can learn cybersecurity in a safe and welcoming environment.

## The longer version

### Expected behaviour
- Be welcoming and inclusive
- Be respectful of differing viewpoints and experience levels
- Give and accept constructive feedback gracefully
- Focus on what's best for the community and for learning
- Show empathy toward others

### Unacceptable behaviour
- Harassment, intimidation, or discrimination of any kind
- Sharing malicious code, exploits, or tools intended to cause harm
- Personal attacks or derogatory comments
- Publishing others' private information without consent
- Gatekeeping — dismissing questions as "too basic" or telling people they don't belong
- Any content that sexualises, endangers, or targets minors

### Scope
This code of conduct applies to all community spaces associated with this project, including GitHub Issues, Discussions, Pull Requests, and any linked community channels.

### Enforcement
Instances of unacceptable behaviour can be reported by opening an Issue with the `conduct` label or by contacting the project maintainers directly. All reports will be reviewed and addressed.
