# Cybersecurity Career Roadmap

Not sure where to start? You're in the right place. Cybersecurity is a massive field with dozens of specialisations — and they all need people. The trick isn't finding the "right" path, it's finding *your* path.

## The paths

Each path below includes: what the role does, skills you need, certifications that matter, resources to learn, and a realistic timeline.

| Path | Starting Role | Where It Leads | Best If You Like... |
|------|--------------|----------------|---------------------|
| [Blue Team](./blue-team.md) | SOC Analyst | Incident Responder → Threat Hunter | Defending, investigating, puzzle-solving |
| [Red Team](./red-team.md) | Junior Pentester | Red Team Operator → Security Consultant | Breaking things (legally), creative thinking |
| [Engineering](./engineering.md) | Security Engineer | Cloud Security Architect → CISO | Building secure systems, automation |
| [GRC](./grc.md) | GRC Analyst | Compliance Manager → Risk Officer | Policy, frameworks, big-picture thinking |
| [Forensics](./forensics.md) | Digital Forensics Analyst | Malware Analyst → Threat Intelligence | Detective work, deep analysis |
| [Bug Bounty](./bug-bounty.md) | Independent Researcher | Full-time Bug Bounty Hunter | Independence, web hacking, self-direction |

## Don't know which path to choose?

That's completely normal. Here's my advice:

1. **Try a bit of everything** — set up a home lab, try a CTF, read about incidents
2. **Notice what excites you** — do you prefer attacking or defending? Policy or code? Fast-paced or methodical?
3. **Talk to people in the field** — join communities, ask questions, attend meetups
4. **You can always switch** — skills transfer between paths more than you'd think

## The universal starting point

No matter which path interests you, everyone benefits from these fundamentals:

- **Networking basics** — TCP/IP, DNS, HTTP, subnetting (CompTIA Network+ level)
- **Linux command line** — navigation, file permissions, basic scripting
- **A scripting language** — Python is the most versatile choice
- **Security fundamentals** — CIA triad, common attack types, defence-in-depth

## Resources that span all paths

- [Hack The Box](https://hackthebox.com) — hands-on labs, slightly more advanced
